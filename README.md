## Getting Started
To run this project locally:

1. Make sure you have XAMPP installed
2. Clone the repository:
   ```
   git clone https://github.com/1ra0ze-art/ozee.git
   ```
3. Move the project folder to your XAMPP `htdocs` directory
4. Start Apache in XAMPP
5. Open your browser and go to `http://localhost/ozee`
